1) создать папку с проектом (новым ботом)

2) проверить наличие папки .vscode и файла .pylintrc 

	.vscode - папка с джисон файлом (допнастройки как проводить импорты, настройки шрифтов, хлебные крошки и тд)
	.pylintrc - файл настроек для пайлинтера (это подсказки при написании кода, pip8, в списоке не нужные напоминалки/ошибки)

3) создать папку venv с виртуальным окружением через терминал

	python -m venv venv           - версия языка 3.10.6 (дефолтная)
	python3.11 -m venv venv       - версия языка 3.11.9

4) запустить виртуальное окружение

	source venv/Scripts/activate  - для GitBash (рекумендуется)
	venv\Scripts\activate         - для Windows

	deactivate                    - выйти из вируального окружения

5) проверить актуальные версии

	python --version
	pip --version


6) скачать необходимые библиотеки из requirements

	pip install -r requirements.txt

   сформировать новый файл requirements

	pip freeze > requirements.txt



7) если будут проблемы с Redis, тогда нужно обновить и переустановить пакеты

	sudo apt update && sudo apt dist-upgrade
	sudo apt install redis

   если не поможет, проверить VPN
   если и это не поможет, в файле main отключить Redis, поменять на MemoryStorage



8) интернализация gettext

        pip install Babel

	pybabel extract --input-dirs=. -o locales/messages.pot
	pybabel extract -o locales/messages.pot --copyright-holder="AdvancedBotsCourse" --project="i18n_example_bot" --version=0.1 --msgid-bugs-address=kms101@yandex.ru --input-dirs=.

	pybabel init -i locales/messages.pot -d locales -D i18n_example_bot -l en
	pybabel init -i locales/messages.pot -d locales -D i18n_example_bot -l ru
	pybabel init -i locales/messages.pot -d locales -D i18n_example_bot -l fr

   скомпилировать файлы переводов в бинарные файлы с расширением .mo
	pybabel compile -d locales -D i18n_example_bot

	pybabel extract -o locales/messages.pot -k _:1,1t -k _:1,2 --copyright-holder="AdvancedBotsCourse" --project="i18n_example_bot" --version=0.1 --msgid-bugs-address=kms101@yandex.ru --input-dirs=.
